Arrow Button Behavior

The arrow buttons feature the classic Windows 10 interaction model:

  • Default: Gray chevron icon (#656565) on light gray background
  • Hover: Background darkens to #e5e5e5, icon turns Windows Blue (#0078d7)
  • Active: Background darkens to #cccccc, icon turns darker blue (#005a9e)
  • Click: Smooth scroll animation (100px per click)

Scrollbar Thumb Behavior

The draggable scrollbar thumb follows Windows 10 conventions:

  • Default: Medium gray color (#c2c3c9) that's subtle but visible
  • Hover: Darkens to #86868b for clear visual feedback
  • Active/Dragging: Dark gray (#555555) shows user control
  • Minimum Height: 40px ensures easy grabbing
  • Proportional Sizing: Thumb size reflects content-to-viewport ratio

Track Behavior

The scrollbar track (the background area) provides context:

  • Light gray background (#f0f0f0) always visible
  • Slightly darker on hover (#e5e5e5) to indicate interactivity
  • Clicking track scrolls content by one page
  • Smooth 0.2s transitions for all state changes

Chevron Arrow Design

The chevron arrows are carefully crafted to match Windows 10:

  • 9px × 9px size fits perfectly in 17px button
  • Clean, angular design with sharp points
  • Centered vertically and horizontally
  • Up arrow points upward, down arrow points downward
  • SVG format ensures crisp rendering at any resolution

Technical Implementation

The scrollbar is built with vanilla JavaScript and CSS, no dependencies required. It uses absolute positioning for the custom scrollbar overlay while hiding the native scrollbar. Event listeners handle scroll synchronization, thumb dragging, and arrow button clicks.

Performance Optimization

The implementation includes several performance optimizations:

  • CSS transitions for smooth 60fps animations
  • Event delegation for efficient memory usage
  • Throttled scroll updates for better performance
  • Hardware-accelerated transforms where possible

Browser Compatibility

Works perfectly in all modern browsers: Chrome, Firefox, Safari, and Edge. The component gracefully degrades - if JavaScript is disabled, the native browser scrollbar appears as fallback.

Accessibility

Full keyboard support is maintained:

  • Arrow keys scroll content
  • Page Up/Down keys work as expected
  • Home/End keys jump to top/bottom
  • Mouse wheel scrolling fully functional
  • Touch/trackpad gestures supported

Usage Instructions

To use this scrollbar in your project:

  1. Copy the HTML structure with .win10-scroll-wrapper
  2. Include the CSS styles in your stylesheet
  3. Add the JavaScript initialization code
  4. Wrap your scrollable content in .scroll-content

The scrollbar automatically calculates thumb size and position based on content height. It responds to window resize events and content changes dynamically.
